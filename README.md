dotfiles
========

Install
-------
    ./install.sh # (-f, --force)

###### If you want use tagbar plugin
    sudo apt-get install ctags

###### If you want use i3
    sudo apt-get install i3 i3lock i3status i3-wm

Useful things
-------------

###### Packages
    sudo apt-get install ack-grep vim-gnome
